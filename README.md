# 비디오 그리드

로컬 동영상 여러 개를 한꺼번에 불러와 **격자(기본 3×2=6분할)로 동시 재생**하고,
동영상별 **시작 시각/이어보기** 옵션을 지정·기억하는 **PWA 앱**. (파일은 그 기기 안에서만 — 서버·클라우드 업로드 없음)

## 설치형 앱(PWA)
- **맥(Chrome/Edge)**: 아래 URL 접속 → 주소창 오른쪽 **설치(⊕)** → 독에 앱 아이콘.
- **안드로이드(Chrome)**: 접속 → 메뉴 → **앱 설치 / 홈 화면에 추가**.
- **공개 URL(폰에서도 접속)**: https://junyoungcha83.github.io/video-grid/

## 로컬 실행(맥에서 직접)
```sh
cd video-grid && python3 -m http.server 8002   # http://127.0.0.1:8002/
```
> 포트 8002(8000=후쿠오카, 8001=secret-gallery 충돌 회피).
> ⚠️ 폰에서 `127.0.0.1`은 폰 자신을 가리킴 → Mac 로컬서버엔 못 들어감. 폰은 위 **공개 URL** 사용.

## 기기별 동작 (중요)
- 이 앱은 **브라우저가 열린 그 기기의 로컬 파일**을 봅니다.
  - 맥에서 열면 → 맥 동영상. 폰에서 열면 → **폰 동영상**(맥 파일 아님).
- **맥(데스크톱)**: `📂 폴더 열기`로 폴더 선택 시 폴더를 **기억**(File System Access API).
- **안드로이드/모바일**: 폴더 기억 미지원 → **`📄 파일 선택`으로 동영상 여러 개 선택**(매번). 모바일은 **동시 재생 개수 제한**이 있어 6개가 다 안 돌 수 있음 → `1×2`/`2×2` 권장.

## 사용법
- **📂 폴더 열기**(Chrome/Edge): 폴더를 한 번 고르면 그 안의 동영상을 모두 불러오고, 폴더를 기억해 다음에 다시 열기 쉬움.
- **📄 파일 선택**(모든 브라우저): 폴더 또는 여러 파일을 직접 선택(매번).
- **배열 2×2 / 3×2 / 3×3**: 격자 칸 수 전환(기본 3×2=6칸, 화면 전체를 채움).
- 칸에 마우스를 올리면 하단 바: **🔇 소리**(클릭=그 영상만 소리), **⚙︎ 설정**, **× 비우기**.
  - 타일 클릭 = 그 영상만 소리 / 더블클릭 = 전체화면.
- **⚙︎ 설정**: **시작 시각**(예 `1:23` 또는 `83`)부터 재생, **이어보기**(닫을 때 위치 기억), 볼륨. 동영상별로 저장됨.
- **🗂️ 목록**: 불러온 동영상 목록에서 빈 칸에 배정.
- 단축키: `H` 바 숨김, `Space` 전체 재생/정지. 마지막 배치·레이아웃은 자동 기억.

## 브라우저 제약
- **폴더 기억(File System Access API)**: Chrome/Edge 전용. 사파리/파이어폭스는 "파일 선택" 폴백 사용.
- 보안상 폴더 경로를 자동으로 못 읽음 → **최초 1회 사용자 선택** 필요(Chrome은 핸들을 기억).
- 소리 자동재생 차단 → 모두 **음소거 자동재생**, 클릭 시 소리(사용자 제스처).
- 6개 동시 디코딩은 PC 성능/코덱에 좌우(H.264 권장). 고해상도·저사양은 끊길 수 있음. MKV/AVI 등은 브라우저가 못 여는 코덱일 수 있음.
